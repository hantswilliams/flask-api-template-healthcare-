Welcome to the Opinionated Healthcare Flask Demo / Template App

This is a simple Flask app that demonstrates how to implement basic security measures that should always be considered when building a healthcare application.

The way this app works, is that we give you a basic user portal as seen here for admins and others with permissions to update specific condfigurations and addition of users. Then there is a series of APIs that are protected using best practices for security (rotating tokens, API limiting, RBAC, etc...). The idea is that this template can then be used to add your own data to using the opioniated security measures and strcture, providing both a administrative GUI in addition to secure API endpoints; or to be used as a security testing tool to try and break the security measures in place.
